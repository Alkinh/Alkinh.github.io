java复习题2

发表于 2021-01-09 | 分类于 笔记

《Java高级程序设计》编程复习题二

1.1、 Java编写程序创建一HashSet集合，并在HashSet集合中添加3个Person对象（属性有id和name），把姓名相同的人当做同一个人，禁止重复添加。然后用数据[01 Tom；xx 张三；02 Tom]测试程序。
（提示：要重写hashCode() 和equals() 方法。）

2.编写一程序，考生利用I/O流（字节流和字符流）或者NIO技术，拷贝项目目录下的一个文件src.txt到dest.txt；再利用字符流技术把dest.txt文件读出来显示在控制台上。（提示：在Java项目的目录下创建源文件src.txt，并输入信息后保存，然后把它拷贝到目标文件和读出。）


3.编写一个JFrame窗口，在窗口最上方放置一个Jlabel标签，标签中默认文本是“此处显示鼠标右键单击坐标”；为Jframe窗口添加一个鼠标事件，当鼠标右键单击窗口时，鼠标的坐标会在Jlabel中显示出来。

4.利用Random类或者Math.random()方法来产生5个3080之间的随机整数。（Tips: nm（n<m）之间的随机数公式: n+(new Random()).nextInt(m-n+1)）

5.编写一Java程序，用中文方式显示当前时间；并输出今天是今年的第几天，星期几。（参考格式为：2020年09月25日：今天是2020年的第269天，星期五）

6.某人在玩游戏时输入密码xxxxxxx(123456)后成功进入游戏（输错5次则被强行退出），要求用程序实现密码验证的过程。（提示： 利用new BufferedReader(new InputStreamReader(System.in))建立由控制台到程序的字符流管道。）
