4.1 数据库索引

1.索引及作用
1.1索引
索引是根据表巾一列或若干列按照一定顺序建立的列值与记录行之间的对应关系表。在列上创建了索引之后，查找数据时可以自接根据该列上的索引找到对应行的位置，从而快速地找到数据。
例如
如果用户创建了学生表中学号列的索引， MySQL将在索引中排序学号列，对于索引中的每一项， MySQL在内部为它保行一个数据文件中实际记录所在位智的“指针”。因此，如果要查找学号为“2019001”的学生信息， MySQL能在学号列的索引中找到“2019001”的值，然后直接转到数据文件中相应的行，准确地返回该行的数据。

1.2索引作用
在数据库系统中建立索引主要有以下作用

  • 快速读取数据
  • 保证数据记录的唯一性
  • 实现表与表之间的参照完整性
  • 在使用 GROUP BY、 ORDER BY子句进行数据检索吋，利用索引可减少排序和分组的时间。

1.3MySQL索引
在 MySQL5.6中，所有的 MySQL列类型都能被索引，但要注意以下几点只有当表类型为 MyISAM、 InnoDB或BDB时，才可以向有NULL、BLOB或TEXT列中添加索引。
一个表最多可有16个索引。最大索引长度足256个字节。
对于CHAR和VARCHAR列，可以索引列的前缀。这样索引的速度吏快并且比索引整个列需要较少的磁盘空间。
MySQL能在多个列上创建索引。索引可以由最多15个列组成（在CHAR和VARCHAR列上，也可以使用列的前缀作为索引的部分）。

2.索引的分类
2.1 BTREE索引目前大部分 MySQL索引都是以B-树（ BTREE）方式存储的，索引类型分成下列几个。

  • （1）普通索引（ INDEX） 这是最基本的索引类型，它没有唯一性之类的限制。创建普通索引的关键字是INDEX。
  • （2）唯一性索引（ UNIQUE) 这种索引和前面的普通索引基本相同，但有一个区别：索引列的所有值都只能出现一次，即必须是唯一的。创建唯一性索引的关键字是 UNIQUE
  • （3）主键（ PRIMARY KEY）主键是种唯性索引，它必须指定为“ PRIMARY KEY”。主键般在创建表的时候指定，也可以通过修改表的方式加入主键。但是每个表只能有一个主键
  • （4）全文索引（ FULLTEXT）MySQL支持全文检索和全文索引。在 My SQL中，个文索引的索引类型为FULLTEXT。

2.2 MySQL索引创建
(1)CREATE|NDEX语句创建使用 CREATE|NDEX语句可以在一个已有表上创建索引，一个表可以创建多个索引。

语法格式
CREATE[UNIQUE| FULLTEXT| SPATIAL]NDEX索引名
[索引类型]
ON表名（索引列名…）
[索引类型]
索引列名=：
列名[(长度)][ASC|DESC]

说明
UNIQUE| FULLTEXT| SPATIAL：∪NQUE表示创建的足唯一性索引； FULLTEXT表示创建全文索引； SPATIAL表示为空间索引，可以用来索引几何数据类型的列（本书不讨论这种索引）。
索引名：索引在一个表中名称必须是唯一的
索引类型：MySQL支持的索引类型有 BTREE和HASH。
索引列名：创建索引的列名后的长度表示该列前面创建索引字符个数。这可使索引文件大大减小，从而节省磁盘空间

4.在创建表时创建索引
在前两种情况下，索引都是在表建立之后创建的。索引也可以在创建表时起创建。在创建表的 CREATE TABLE语句中可以包含索引的定义。
语法格式
create [TEMPORARY] TABLE [IF NOT EXISTS] 表名
[([列定义],…|[索引定义])]
[表选项][select语句];
索引定义二：
[CONSTRAINT [symbol]]PRIMARY KEY [索引类型](索引列名…） /主键/
|{INDEX|KEY}[索引名][索引类型](索引列名…） /索引/
|[CONSTRAINT[symonl]] UNIQUE [INDEX|KEY][索引名][索引类型](索引列名…） /唯一性索引/
|[FULLTEXT|SPATIAL][INDEX|KEY][索引名](索引列名…） /全文索引/
|[CONSTRAINT[symbol]] FOREIGN KEY 索引名[参照系定义] /外键/

5.删除索引
当个索引不再需要的吋候，可以用DROP INDEX语句或 ALTER TABLE语句删除它。
(1) 使用DROP INDEX删除
语法格式：
DROP IDNEX 索引名 ON 表名

(2)使用 ALTER TABLE删除语法格式：
语法格式：
ALTER [IGNORE] TABLE 表名
…..
| DROP PRIMARY KEY /删除主键/
| DROP {INDEX|KEY}索引名 /删除所有/
| DROP FOREIGN KEY fk_symbol /删除外键/
