章节3.3 数据修改

1.修改单个表
语法格式：
update 表名
set 列名=expr1 [，列名2=expr2…]
[where 条件]
[order by]

示例1
将学生成绩数据库 stumarkdb的课程表中的所有课程的学时都加1，将课程所开设的系别改为“计算机信息工程学院”。

实例2
将学生表中的张三姓名改为张锋，电话改为19023456789

2.改多个表
语法格式
update 表名，表名
set 列名1=exp1[,列名2=exp2…]
[where 条件]

示例1
将 stumarkdb数据库中的tb_course的学分设置成和testcourse表中的学分一样.（类似复制黏贴)
