章节3.1

1.sql语言
结构化查询语言简称SQL语言，是一种特殊目的的编程语言，是一种数据库查询和程序设计语言，用于查询数据及查询，更新和管理关系数据库系统。

2.sql语言功能上分三部分

  • DDL:数据定义语句
  • DML:数据操纵语言
  • DCL：数据控制语句

2.1DDL
数据定义语言（DDL）。用于执行数据库的任务，对数据库及数据库中的各种对象进行创建、删除、修改等操作。如前所述，数据库对象主要包括：表默认约束、规则、视图、触发器、存储过程等。DDL包括的主要语句及功能如表所示。

语句 功能 说明
create 创建数据库及数据库对象 不同数据库对象，其create语句的语法形式不同
alter 对数据库或数据库对象进行修改 不同数据库对象，其alter语句的语法形式不同
drop 删除数据库或数据库对象 不同数据库对象，其drop语句的语法形式不同

2.2DML
数据操纵语言（DML）。用于操纵数据库中各种对象，检索和修改数据。DML包括的主要语句及功能如表所示。

语句 功能 说明
SELECI 从表或视图中检索数据 是使用最频繁的SQL语句之一
INSERT 数据插入到表或视图中
UPDATE 修改表或视图中的数据 既可修改表或视图的，也可修改一组或全部数据
DELETE 从表或视图中删除数据 可根据条件删除指定的数据

2.3DCL
数据控制语言（DCL）。用于安全管理，确定那些用户可以查看或修改数据库中的数据，DCL包括的主要语句及功能如表所示。

语句 功能 说明
grant 授予权限 可把语句许可或对象许可的权限授予其他用户和角色
revoke 收回权限 收回权限与grant的功能相反，但不影响该用户或角色从其他角色中作为成员继承许可权限

2.4
My SQL增加的语言元素。这部分不是SQL标准所包含的内容，而是为了用户编程的方便增加的语言元素。这些语言元素包括常量、变量、运算符、函数、流程控制语句和注解等。

3.1
mysql的数据类型包括：

  • 数值类型
    • 整数类型
    • 浮点类型
    • 定点数型
  • 字符串类型
  • 日期时间类型

3.1.1 整数类型
整数类型的划分是根据存储空间的大小和表示范围的大小来划分的，具体如下表所示

数据类型 存储空间（Bytes） 表示范围
TINY INT 1 -128~127
SMALL INT 2 -32768~32767
MEDIUM INT 3 -8388608~8388607
INT 4 -2147483648~2147483647
BIG INT 8 ±9.22×10的18次方

3.1.2 浮点数类型
浮点数类型是用来表示实数的一种方法，根据位数和精度，浮点数类型又分为单精（ FLOAT）浮点数和双精浮点数（ DOUBLE），具体如下表所示。

数据类型 存储空间 精度
FLOAT（M, D） 4 8位精度，M表示数字的总位数，D表示小数点后面数字的位数
DOUBLE（M, D） 8 16位精度，M表示数字的总位数，D表示小数点后面数字的位数

3.1.2字符串类型

  • char：固定长度字符串
  • varchar：可变长度字符串
  • Binary：无字符集的字符串
  • Varbinary：可变长无字符集的字符串
  • Blob和text：以对象类型保存，足够大的数据
  • Enum：枚举类型，从值集合中选取单个值
  • Sct：集合对象，从值集合中迭取多个值
