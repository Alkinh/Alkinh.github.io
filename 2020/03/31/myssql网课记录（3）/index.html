myssql网课记录（3）

发表于 2020-03-31 | 分类于 网课记录

#章节2.3

1.添加用户
create user “用户名”@”主机名(本地一般是localhost）” identified by “密码”;

2.授予用户权限
grant privileges on 数据库名.表名 to 用户
其中 privileges 可以包含
create（创建） alter（改变） drop（删除）： 库，表结构操作
select（选择） insert(插入） delete（删除） update（更新）：数据操作
references:外键权限
index:索引

演示
为用户xm授予STUMARKDB数据库中所有表的select、insert、create及alter的权利。

3.删除用户的权限
revoke privileges(要回收的权限) on 数据库名.表名 from 用户

演示
删除用户xm的select权限

4.修改用户名
rename user ‘老用户名‘@’localhost’ to ‘新用户名‘@’localhost’;

5.修改用户密码
set password for’用户名‘@’localhost’=password(‘密码’);

6.删除用户
drop user ‘用户名‘@’localhost’;
